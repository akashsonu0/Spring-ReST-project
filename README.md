# Spring-"ReST-project"

A complete hands-on **Spring Boot REST API practice repository** containing multiple real-world implementations of RESTful service development.  
This project includes end-to-end exercises using **Request Handling, ResponseEntity, Exception Handling, WebClient, RestTemplate, Redis Cache, H2 Database, and Unit Testing (JUnit + Mockito).**  
Perfect for mastering backend development & microservices communication.
